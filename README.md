# Linas Spielwiese

## Steuerung
- **W A S D**: laufen
- **Mouse**: umsehen
- **Shift**: sprint
- **E**: Interaktion (bei Raum-Schildern teleportiert dich E in den Raum)
- **Click** (während Pointer Lock aktiv): Projekt öffnen (Modal)
- **Esc**: Maus freigeben / Overlay

## Hinweis
Das Projekt ist komplett ohne externe Libraries/ohne CDN gebaut (reines WebGL2).
